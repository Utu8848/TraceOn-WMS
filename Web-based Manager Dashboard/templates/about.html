Project Introduction

TraceOn Our team is focused on improving workplace productivity and accountability by developing a system that monitors and manages digital activity in an ethical and effective way. Our project is a Work Monitoring System called TraceOn that helps organizations monitor and evaluate employee productivity in real-time. The system automatically captures periodic screenshots, tracks application and website usage, and presents this data in a secure dashboard for managerial review. Additionally, Generates productivity report by analyzing the screenshots.

In remote and hybrid workplaces, managers often struggle to ensure accountability and measure actual work being done. This can lead to reduced productivity, time misuse, and a lack of transparency. Our system addresses this by offering a non-intrusive yet effective way to track employee activity, ensuring fairness, improving performance, and helping teams stay aligned.

Design & Solution

Innovative approach to remote work monitoring that balances productivity tracking with employee privacy protection

Challenge Analysis

The primary challenge addressed by TraceOn is the delicate balance between effective remote work monitoring and employee privacy protection.

Traditional monitoring solutions often:


  • Employ invasive surveillance techniques
  • Fail to provide contextual work insights
  • Breach employee privacy and trust
  • Lack intelligent activity detection

Proposed Solution

TraceOn implements a hybrid approach combining periodic and activity-triggered screenshot capture with intelligent privacy filtering.

Our solution features:


  • OCR-based sensitive information redaction
  • Context-aware activity monitoring
  • Secure cloud-based storage with Google Drive integration
  • Role-based access control for managers
  • Automated folder organization and data management

Use Case

TraceOn serves remote teams, hybrid organizations, and distributed workforces requiring accountability measures without compromising employee privacy.

The system is particularly valuable for:


  • Project-based teams needing progress tracking
  • Organizations with compliance requirements
  • Companies transitioning to remote work models
  • Managers overseeing distributed teams

Prototype Description

Our Minimum Viable Product (MVP) demonstrates core functionality with a two-layer architecture

Worker Application

Desktop application with GUI for employee registration, authentication, and real-time activity monitoring with privacy-aware screenshot capture.

Manager Dashboard

Web-based interface for user management, authentication, and secure access to organized screenshot data stored in cloud folders.

Key Features

Authentication System

Secure login with password hashing and role verification

Activity Monitoring

Detects new windows, tabs, and domain changes using pygetwindow and psutil

Privacy Protection

OCR-based filtering removes sensitive information before storage

Cloud Integration

Automated hierarchical folder creation and management

Access Control

Role-based permissions with Google Drive integration
